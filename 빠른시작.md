## 🎯 가장 인기 있는 3가지

### 1. Product Swiper (제품 소개)

```tsx
import { ProductSwiper } from "./components";

const products = [
  { image: "1.jpg", title: "제품1", description: "설명..." },
  { image: "2.jpg", title: "제품2", description: "설명..." }
];

<ProductSwiper slides={products} stickyMode={true} gradientOverlay={true} />;
```

**언제 사용?**

- 제품 소개 페이지
- 포트폴리오
- 서비스 소개

---

### 2. Sticky Image Zoom (임팩트 이미지)

```tsx
import { StickyImageZoom } from "./components";

<StickyImageZoom
  imageSrc="hero.jpg"
  initialScale={0.1}
  finalScale={1}
  overlayContent={<h1>브랜드명</h1>}
/>;
```

**언제 사용?**

- 메인 히어로 섹션
- 브랜드 소개
- 임팩트 필요할 때

---

### 3. Section Navigation (쇼핑몰 네비)

```tsx
import { SectionNavigation } from './components';

<SectionNavigation
  sections={[
    { id: "info", label: "상품정보" },
    { id: "review", label: "리뷰" },
    { id: "qna", label: "Q&A" }
  ]}
/>

<section id="info">...</section>
<section id="review">...</section>
<section id="qna">...</section>
```

**언제 사용?**

- 쇼핑몰 상세 페이지
- 리뷰 페이지
- 긴 페이지 네비게이션

## 💡 조합 예제
