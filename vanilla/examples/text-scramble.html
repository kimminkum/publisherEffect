⚡ Text Scramble Effect

해커/사이버펑크 스타일의 텍스트 스크램블 효과

1. Auto Trigger (자동 실행)

CYBER PUNK STYLE 
new TextScramble(element, {
  trigger: 'auto',
  speed: 50,
  delay: 0
});

2. Click Trigger (클릭 시 실행)

Click Me to Scramble! 
new TextScramble(element, {
  trigger: 'click',
  speed: 40
});

3. Hover Trigger (마우스 올릴 때 실행)

Hover Over Me! 
new TextScramble(element, {
  trigger: 'hover',
  speed: 30
});

4. Custom Characters (커스텀 문자)

CUSTOM CHARS 
new TextScramble(element, {
  trigger: 'auto',
  characters: '01',  // 0과 1만 사용
  speed: 30,
  delay: 1000
});
